# Requirement

- node.js
- vue cli
- vue-router

# Installation

```bash
npm install -g vue-cli
npm install vue-router
```

# Usage
## Compiles and hot-reloads for development

```bash
npm run serve
```

## Compiles and minifies for production
```bash
npm run build
```
